# Customer Sync Refactoring Kata in Typescript

## Getting started

To install dependencies with NPM:

```
npm install
```

To install dependencies with YARN:

```
yarn install
```

## Running tests

With NPM:

```
npm run test:mocha
```

With YARN:

```
yarn test:mocha
```

## Generating html coverage output

With NPM

```
npm run nyc:report
```

With YARN:

```
yarn nyc:report
```
